# Skills.sh Platform Information

## What is Skills.sh?

Skills.sh 是一个 AI 技能开发和分发的标准化平台，专注于为 CatPaw AI 和其他 AI 助手提供统一的技能配置标准。

## Platform Features

### Core Capabilities
- **技能标准化**: 统一的 SKILL.md 格式规范
- **技能分发**: 便捷的分享和部署机制
- **技能发现**: 分类浏览和搜索功能
- **技能验证**: 质量检查和认证机制

### Supported Platforms
- CatPaw AI 助手
- Claude Code IDE
- Cursor IDE
- OpenClaw 生态系统
- Trae IDE
- VS Code 扩展

## Getting Started with Skills.sh

### For Users

