# Dioxus Element Plug Skills

这是一个专为 Dioxus Element Plug 项目设计的 CatPaw AI 技能配置文件项目。

## 项目概述

本项目包含 `skills.sh` 配置文件，为 CatPaw AI 助手提供专业的 Dioxus Element Plug 项目技能支持，专注于：

- 🚀 Modern Dioxus 0.7+ UI 开发
- 🎨 Element UI 设计系统实现
- ⚡ 零配置 SCSS 支持
- 📦 完整的 crates.io 包管理
- 🌟 开发者体验优化

## 核心功能

### AI 技能配置
- **项目识别**: 支持 Dioxus Element Plug 项目的深度理解
- **技术栈指导**: 提供完整的技术栈配置建议
- **问题解决**: 常见问题的诊断和解决方案
- **最佳实践**: 现代 DX 原则贯穿始终

### 技能特性
- **零配置哲学**: 开箱即用的开发体验
- **热重载支持**: 即时反馈的视觉开发
- **编译时优化**: 类型安全和资源处理的保证
- **渐进式增强**: 简单起步，易于扩展

## 使用方法

1. 将此项目克隆到本地
2. 将 `skills.sh` 文件复制到 CatPaw 的 AI 技能目录
3. 重启 CatPaw 以加载新技能

## 配置内容

### 响应行为
- 提供完整的、可复制的代码示例
- 分步实施指南
- 最佳实践和现代模式
- 问题诊断和解决方案路径

### 教学方式
- 概念优先：先解释理论再展示代码
- 示例驱动：提供多个实际示例
- 渐进式：从简单开始，逐步增加复杂度
- 情境化：将新概念与熟悉的概念关联
